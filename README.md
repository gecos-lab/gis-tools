# gis-tools

__Useful tools for GIS.__

Generally QGis scripts must be placed in:
C:\Users\<your_user_name>\AppData\Roaming\QGIS\QGIS3\profiles\default\processing\scripts

_Legend scripts used to automatically convert QGis legends for point and polygon layers, keeping the same RGB color scheme.:_

- point2polygon_legend.py
- polygon2point_legend.py
- point2polygon_rule_legend.py
- polygon2point_rule_legend.py
